# 背景音乐说明

## 游戏音乐
- [MUST] background_music.mp3 - 主界面背景音乐
- [MUST] battle_music.mp3 - 战斗背景音乐
- shop_music.mp3 - 商店背景音乐
- victory_music.mp3 - 胜利音乐
- evolution_music.mp3 - 进化音乐

音乐要求：
1. 必须使用MP3格式
2. 比特率：128kbps或192kbps
3. 采样率：44.1kHz
4. 声道：立体声
5. 时长：建议30秒-3分钟
6. 文件大小：建议小于5MB
7. 必须支持循环播放
